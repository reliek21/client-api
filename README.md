
# Client API

Link: https://client-api-21.herokuapp.com/


## Run Locally

Clone the project

```bash
  git clone https://github.com/reliek21/client-api.git
```

Go to the project directory

```bash
  cd client-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
